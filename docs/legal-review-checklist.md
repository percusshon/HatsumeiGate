## 2. 法的判断が必要な確認事項（弁護士確認推奨）

- **特定商取引法に基づく表記**の要否。手数料（成立取引の原則20%）や取引形態が
  特商法の規制対象に当たるかを確認し、必要なら表記ページを追加する。
- **責任制限条項（利用規約 第13条）の上限額**と、消費者契約法（事業者の責任を
  不当に免除する条項の無効）との整合。
- **外国にある第三者への提供・委託**（プライバシー 第5条）: 委託先クラウド事業者が
  外国にある場合の APPI 上の対応（本人への情報提供／同意取得の要否）を確定する。
  保存リージョンは東京を指定する方針だが、事業者の所在は国外であり得る。
- **手数料率の表記**（利用規約 第10条）。実装の既定は20%（operator 上書き可、
  `lib/revenue/events.ts` の `DEFAULT_PLATFORM_FEE_RATE`）。規約の表現と運用を一致させる。
- 反社会的勢力排除条項・準拠法・合意管轄の最終文言。
- 知的財産権の帰属（利用規約 第7条）と、当社の投稿情報の利用範囲の表現。

## 3. 実装との整合（記載済みの事実）

公開文書は以下の実装事実に基づいて記載しているため、実装変更時は文書も見直すこと。

- 認証: メールのログインリンク（パスワードレス、`app/login/actions.ts` の `signInWithOtp`）。
- 収集項目: `users_profile`（email/表示名/role/言語/TZ）、投稿情報・ファイル、NDA・開示・取引、
  監査ログ（`audit_logs`／`company_invention_views` の IP・UA）。
- 安全管理: RLS＋アプリ認可、service_role はサーバー限定、開示ファイルの表示時透かし、閲覧記録。
- 委託先: Supabase（DB/認証/ストレージ）、Vercel（ホスティング）、認証メール送信サービス。
- 手数料: 既定20%、operator 上書き可。
